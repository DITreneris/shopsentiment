ShopSentiment Documentation

Introduction

ShopSentiment is a web application designed to help users analyze product reviews from e-commerce platforms. The application performs sentiment analysis on reviews, extracts key themes and topics, and presents the data in an intuitive dashboard format.

Key Features

  • Sentiment analysis of product reviews (positive, neutral, negative)
  • Keyword extraction to identify common themes
  • Sentiment trends over time
  • Interactive filtering and visualization
  • Data export functionality
  • User account management
  • Light and dark theme options

Getting Started

To begin using ShopSentiment, follow these steps:

  1. Create an account: Click the "Register" button in the navigation bar to create a new account. This will allow you to save your analyses and access them later.
  2. Log in: After creating an account, log in using your credentials.
  3. Navigate to the home page: The home page provides access to the product analysis form.

Analyzing Products

To analyze a product's reviews:

  1. Select a platform: Choose the e-commerce platform where the product is listed (Amazon, eBay, or Custom).
  2. Enter product ID or URL: Provide the product ID or full URL from the chosen platform.
  3. Submit for analysis: Click the "Analyze" button to begin the review collection and analysis process.
  4. Wait for results: The analysis may take a few moments to complete as reviews are collected and processed.

For custom platforms, additional configuration options will be available to specify how to locate reviews on the page.

Understanding Results

The analysis dashboard consists of several key components:

Sentiment Distribution

A pie chart showing the proportion of positive, neutral, and negative reviews. This gives you a quick overview of overall customer sentiment.

Top Keywords

A bar chart displaying the most frequently mentioned words in reviews. This helps identify common themes or features that customers are discussing.

Sentiment Trend

A line graph showing how sentiment has changed over time. This can help identify if product perception is improving or declining.

Reviews Table

A detailed table of all collected reviews, including the original text, rating, date, and calculated sentiment score.

Filtering Options

Use the filter controls to focus on specific sentiments or keywords, helping you drill down into particular aspects of customer feedback.

User Account

Your user account provides several benefits:

Profile Management

Access your profile by clicking on your username in the navigation bar. From here, you can update your username and email address.

Saved Analyses

All analyses you perform while logged in are saved to your account. Access them from the "My Analyses" section in the navigation menu.

Privacy

Your analyses are private and can only be accessed by you when logged into your account.

Theme Options

ShopSentiment offers both light and dark themes:

Switching Themes

Toggle between light and dark themes using the theme switch in the navigation bar (sun/moon icon).

Persistence

Your theme preference is saved to your browser's local storage and will persist between sessions.

Data Export

Export your analysis data for further processing or reporting:

CSV Export

Click the "Export CSV" button to download a CSV file containing all review data, including text, ratings, dates, and sentiment scores.

JSON Export

Click the "Export JSON" button to download a JSON file containing all review data, along with product information and metadata.

Troubleshooting

Common issues and their solutions:

No Reviews Found

If no reviews are found for a product, check that the product ID or URL is correct. Some products may have reviews disabled or use a non-standard format.

Custom Platform Issues

For custom platforms, ensure that the CSS selectors are correctly defined to locate review elements on the page.

Login Problems

If you're having trouble logging in, use the password reset feature or contact support for assistance.
